### Sobre o projeto

O projeto consiste na construção de uma página web para o **Laboratório de Ensino e Pesquisa em Farmácia Social (LEPFS)**, com o intuito de mostrar informações atualizadas sobre o laboratório, como membros, missão, ações, depoimentos, entre outros.
