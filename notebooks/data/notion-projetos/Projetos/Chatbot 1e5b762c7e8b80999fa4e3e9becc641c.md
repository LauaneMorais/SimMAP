### Sobre o projeto

O projeto consiste em uma plataforma para o Processo Seletivo da LAWD 2025, onde os candidatos irão se inscrever e os avaliadores poderão disponibilizar os horários das entrevistas, efetuar anotações para as mesmas e gerar notas para cada candidato, a fim de automatizar o processo.
